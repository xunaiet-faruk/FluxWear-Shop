# 🛍️ Urban Style - Fashion E-commerce

A modern, responsive fashion e-commerce frontend built with **React + Vite**. This project showcases a clean shopping experience using local dummy data, featuring product browsing, filtering, sorting, product details, and a fully functional shopping cart.

> 🚀 Developed as a Frontend Developer assessment project.

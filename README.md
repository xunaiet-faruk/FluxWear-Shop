# 🛍️ Urban Style - Fashion E-commerce

A modern, responsive fashion e-commerce frontend built with **React + Vite**. This project showcases a clean shopping experience using local dummy data, featuring product browsing, filtering, sorting, product details, and a fully functional shopping cart.

> 🚀 Developed as a Frontend Developer assessment project.

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/xunaiet-faruk/FluxWear-Shop
```

---

## Navigate to the project

```bash
cd your-project-folder
```

---

## Install dependencies

```bash
npm install
```

---

## Start development server

```bash
npm run dev
```

---

## Build Project

```bash
npm run build
```

# 📦 Dummy Data

This project uses **local dummy product data** (10–12 products).

Each product includes:

- ID
- Name
- Category
- Price
- Image
- Rating
- Colors
- Sizes
- Stock Status
- Description

No backend or external database is used.

# FluxWear - A modern and responsive fashion e-commerce frontend built with React and Vite. The project uses local dummy data and focuses on clean UI, reusable components, and a smooth shopping experience.


## Live Demo
[FluxWear](https://flux-wear.vercel.app)

## GitHub Repository
[FluxWear-Shop](https://github.com/xunaiet-faruk/FluxWear-Shop)

## Tech Stack
- React.js
- Vite
- React Router DOM
- Tailwind CSS
- React Icons
- Context API

## ✨ Features

- 🛍️ Browse products
- 🔍 Product details page
- 🛒 Shopping cart with Context API
- 📱 Fully responsive (Mobile, Tablet & Desktop)
- ⚡ Loading & Empty states
- 🎨 Modern and interactive UI
- 🔄 Client-side routing

## Installation
```bash
git clone https://github.com/xunaiet-faruk/FluxWear-Shop.git
cd FluxWear-Shop
npm install
npm run dev
